# ***Fino World***
Video Library built with ReactJS which contains information regarding Finance and Investments
</div>

## Installation

Install and Run Locally in 4 simple steps:

```bash
  $ git clone https://github.com/srv-pranjal/Fino_World.git
  $ cd Fino_World
  $ npm install
  $ npm start
```

## Features

- Home Page
- Video-Listing Page
- Filter Videos by Categories
- Watch Video
- Like Video
- Share Video URL
- Playlist Management
  - Add to Playlist
  - Remove from Playlist
  - Create New Playlist
  - Delete Playlist
- WatchLater Management
  - Add to WatchLater
  - Remove From WatchLater
- History
- Authentication
  - Login
  - Signup
  - Logout
  - Forgot Password
- Fully Responsive

## Built With

- ReactJS
- React Router v6
- Vanilla CSS integrated with Sapphire-UI Component Library
- EmailJS
